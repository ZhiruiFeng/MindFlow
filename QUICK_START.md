## ✅ 检查清单

## 🎯 三步快速启动

### Step 1: 创建 Xcode 项目（3 分钟）

```bash
# 1. 确保在项目根目录
cd /Users/zhiruifeng/Workspace/dev/MindFlow

# 2. 打开 Xcode
open -a Xcode
```

在 Xcode 中：

1. **File** → **New** → **Project**
2. 选择 **macOS** → **App**
3. 配置：
   - Product Name: `MindFlow`
   - Interface: `SwiftUI`
   - Language: `Swift`
4. 保存到当前目录（选择 MindFlow 文件夹的上层目录）

### Step 2: 配置项目（1 分钟）

在 Xcode 中：

1. **删除默认文件**：
   - 删除自动生成的 `ContentView.swift`

2. **导入项目文件**：
   - 将 `MindFlow/` 目录下的所有文件夹拖入 Xcode
   - 勾选 `Copy items if needed`

3. **配置权限**：
   - 选择项目 → Target → Info
   - 添加两个权限描述：
     - `Privacy - Microphone Usage Description`: "MindFlow 需要访问麦克风以录制您的语音并转换为文字。"
     - `Privacy - AppleEvents Sending Usage Description`: "MindFlow 需要发送键盘事件以实现自动粘贴功能。"

4. **配置为菜单栏应用**：
   - 在 Info 中添加：
     - `Application is agent (UIElement)`: `YES`

### Step 3: 运行应用（1 分钟）

1. 选择 **My Mac** 作为运行目标
2. 点击 ▶️ **Run**（或按 `⌘R`）
3. 授予**麦克风权限**（必需）
4. 授予**辅助功能权限**（可选，用于自动粘贴）

## 🎤 立即体验

### 第一次使用

1. **配置 API Key**
   - 点击菜单栏的 🎤 图标
   - 选择 `设置...`
   - 输入你的 OpenAI API Key
   - 点击 `保存`

2. **开始录音**
   - 点击菜单栏图标 → `开始录音`
   - 或按快捷键 `⌘ Shift V`

3. **说话**
   - 对着麦克风清晰地说一句话
   - 例如："嗯，那个，我想说的是，今天天气真好"

4. **停止并查看结果**
   - 点击 `停止并处理`
   - 等待几秒，会显示：
     - **原始文本**: "嗯，那个，我想说的是，今天天气真好"
     - **优化后**: "我想说的是，今天天气真好。"

5. **使用文本**
   - 点击 `粘贴` 按钮，文本会自动插入到你的活动应用中
   - 或点击 `复制` 后手动粘贴

## 💡 使用场景

### 场景 1: 快速写邮件

1. 打开邮件应用，点击到正文输入框
2. 按 `⌘ Shift V`
3. 说出邮件内容（口语化也没关系）
4. 点击粘贴，AI 优化后的文本会自动插入

### 场景 2: 记笔记

1. 在任何笔记应用（Notion, Obsidian, Apple Notes 等）
2. 按 `⌘ Shift V`
3. 说出想法
4. 自动转换为整洁的文字

### 场景 3: 聊天消息

1. 在微信、Slack、Discord 等聊天应用
2. 按 `⌘ Shift V`
3. 说出要发送的消息
4. AI 自动润色，让你的消息更专业

## ⚙️ 优化级别说明

在设置或预览界面，你可以选择三种优化级别：

| 级别 | 效果 | 适用场景 |
|------|------|----------|
| **轻度** | 仅去除明显填充词，保留口语风格 | 聊天、即时消息 |
| **中度** | 去除填充词 + 优化语句结构 | 邮件、工作文档 |
| **重度** | 深度改写，转换为书面化 | 正式报告、演讲稿 |

**示例**：

原文：
> "嗯，那个，我想说的是，就是这个项目，呃，需要在下周之前完成，然后还要做测试"

**轻度优化**：
> "我想说的是，这个项目需要在下周之前完成，然后还要做测试。"

**中度优化**：
> "这个项目需要在下周之前完成，并进行测试。"

**重度优化**：
> "该项目需在下周前完成开发与测试工作。"

## 🐛 遇到问题？

### 录音没反应
- ✅ 检查麦克风权限（系统设置 → 隐私与安全性 → 麦克风）
- ✅ 检查麦克风是否被其他应用占用
- ✅ 重启应用

### API 调用失败
- ✅ 检查 API Key 是否正确
- ✅ 检查网络连接（需要访问 api.openai.com）
- ✅ 检查 OpenAI 账户余额
- ✅ 查看 Xcode 控制台的错误日志

### 热键不工作
- ✅ 授予辅助功能权限
- ✅ 检查是否有其他应用占用了相同的热键
- ✅ 重启应用

### 自动粘贴不工作
- ✅ 授予辅助功能权限（必需）
- ✅ 可以先用"复制"功能，然后手动粘贴
